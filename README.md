# Blog Post Page

This repository contains my solution to the [Blog Post Page](https://roadmap.sh/projects/blog-post-page) challenge on roadmap.sh.

## Project Details
The objective of this challenge is to structure long-form content using valid, semantic HTML5. The project focuses on using appropriate document landmarks to improve SEO, accessibility, and the overall maintainability of the codebase.

## Requirements Met
- **Semantic Page Structure:** - Uses `<header>`, `<main>`, and `<footer>` for page-level navigation and identity.
    - Encapsulates the core blog content within an `<article>` tag.
    - Groups content sub-sections using the `<section>` element.
- **Accessible Content Tags:**
    - Correctly implements `<figure>` and `<figcaption>` to semantically associate images with their descriptions.
    - Uses `<time>` with `datetime` attributes for machine-readable dates.
    - Implements proper `<blockquote>` and `<cite>` structures for quoted references.
- **Code Representation:** - Employs `<pre>` and `<code>` in combination to display code blocks while preserving exact whitespace and formatting.
- **Head Metadata:** Includes essential `<meta charset>`, `<meta viewport>`, and `<title>` tags to ensure proper browser rendering and search engine indexing.
